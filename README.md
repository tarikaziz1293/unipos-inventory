# UniPOS Inventory Management

A full-featured inventory dashboard built with React + Recharts.

## Features
- Product catalogue with colour × size variations
- Stock level tracking & alerts
- Print Catalogue → downloads full stock report as HTML (open in browser → Ctrl+P → Save as PDF)
- Charts: monthly stock in/out, by category, by size
- Add / Edit / Delete products with image uploads

## Setup

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Deploy to Vercel
1. Push this folder to a GitHub repo
2. Go to vercel.com → New Project → import your repo
3. Click Deploy — done!
